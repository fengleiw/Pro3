THÔNG SỐ SẢN PHẨM HOLLOW KNIGHT REMASTER

1. Thông số kỹ thuật:
- Kích thước file build: 156MB
- Phiên bản Unity: 2022.3.16f1
- Định dạng: Windows x64 (.exe)
- Yêu cầu hệ thống tối thiểu:
  + CPU: Intel Core i3 hoặc tương đương
  + RAM: 4GB
  + GPU: Intel HD Graphics 4000 hoặc tương đương
  + Dung lượng ổ cứng: 200MB
  + Hệ điều hành: Windows 7 SP1 trở lên

2. Kết quả test hiệu suất:
- FPS trung bình: 60 FPS (ổn định)
- Thời gian tải màn chơi: 2-3 giây
- Sử dụng RAM: ~500MB
- Sử dụng CPU: 15-20%
- Sử dụng GPU: 30-45%
  + GPU Memory: ~200MB
  + GPU Power: 25-35W
  + GPU Temperature: 45-55°C
- Độ trễ input: <16ms
- Độ trễ GPU: 8-12ms (trung bình 10ms)
  + Render time: 6-8ms
  + Present time: 2-4ms
  + Không có hiện tượng frame drop

3. Thống kê nội dung:
- Số lượng màn chơi: 5
- Số lượng kẻ địch: 8 loại
- Số lượng boss: 1
- Số lượng kỹ năng nhân vật: 6
- Thời gian chơi ước tính: 2-3 giờ

4. Kết quả test tương thích:
- Windows 10: Hoạt động tốt
- Windows 11: Hoạt động tốt
- Windows 7: Có một số vấn đề với hiệu ứng đồ họa
- Độ phân giải màn hình hỗ trợ: 1280x720 đến 1920x1080

5. Các vấn đề đã phát hiện:
- Đôi khi có hiện tượng giật lag nhẹ khi chuyển màn
- Một số hiệu ứng âm thanh có thể bị chồng chéo
- Có thể gặp lỗi khi chuyển tab game quá nhanh

6. Đánh giá tổng quan:
- Độ ổn định: 9/10
- Hiệu suất: 8/10
- Tương thích: 9/10
- Trải nghiệm người dùng: 8.5/10
